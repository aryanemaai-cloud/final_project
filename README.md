# Simple Interest Calculator

This project is a simple interest calculator implemented using a Bash script.

## 📌 Description
The calculator takes user input for:
- Principal amount
- Rate of interest
- Time period

It then calculates the simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## ⚙️ How to Use

1. Run the script:
   ./simple-interest.sh

2. Enter:
   - Principal amount
   - Rate of interest
   - Time period

3. The calculated simple interest will be displayed.

## 💡 Example

If:
- Principal = 1000  
- Rate = 5  
- Time = 2  

Then:
Simple Interest = 100
